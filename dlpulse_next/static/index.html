    Open source thanks

    DLPulse Next builds on the work of many open-source projects and communities. Thank you to their authors, maintainers, and contributors.

    Projects we rely on

    • yt-dlp — media extraction and downloads from many sites
    • FFmpeg — muxing, remux, and audio conversion (bundled via imageio-ffmpeg or system install)
    • Flask & Werkzeug — local HTTP API and UI serving
    • pywebview — native desktop window (Linux: system WebKitGTK + GTK3)
    • PyChromecast & Zeroconf — Chromecast discovery and control
    • Mutagen — audio metadata
    • imageio-ffmpeg — FFmpeg binaries for desktop bundles
    • aria2c — optional multi-connection download accelerator
    • qrcode — donation QR codes
    • packaging — version parsing
    • Python — runtime and ecosystem (plus transitive libraries)

    Often used alongside

    • mpv & VLC — optional external players (Settings)
    • HTML5 <video> — in-app playback via the embedded browser engine
    • WebKitGTK / GTK — system packages on Linux (see README); WebView2 / WKWebView on Windows / macOS

    If you maintain one of these projects: thank you. Bug reports and patches upstream help everyone.

    About DLPulse

    DLPulse Next is the desktop edition of the DLPulse ecosystem — cross-platform tools around yt-dlp for search, downloads, playback, and Chromecast. On the project site you can compare editions (Flet, CLI, Textual, Android) and browse releases.

    Project site
